Lógica del Programa

Un número primo es aquel que solo es divisible por sí mismo y por 1. El programa recorre cada número en el rango que proporcionas. Para cada número, una función auxiliar verifica sus divisores. Por eficiencia, esta verificación se hace solo hasta la raíz cuadrada del número, lo que acelera mucho el proceso.

// Función auxiliar que determina si un número es primo.
const esPrimo = num => {
    // Los números menores o iguales a 1 no son primos.
    if (num <= 1) return false;

    // Se itera desde 2 hasta la raíz cuadrada del número.
    // Si se encuentra un divisor en este rango, no es primo.
    for (let i = 2; i * i <= num; i++) {
        if (num % i === 0) return false;
    }

    // Si el bucle termina sin encontrar divisores, el número es primo.
    return true;
};

Lógica del Programa

La serie Fibonacci comienza con 0 y 1. Cada número subsiguiente es la suma de los dos anteriores (0, 1, 1, 2, 3, 5, 8...). La función crea una lista con los dos primeros valores. Luego, un bucle calcula el siguiente número y, si no excede el límite que indicaste, lo agrega a la lista y repite el proceso.

function ejecutarFibonacci() {
    const limite = parseInt(document.getElementById('fibonacciLimite').value);
    
    // Se inicia la secuencia con los dos primeros números.
    let secuencia = [0, 1];

    // Caso especial: si el límite es 0, la secuencia es solo [0].
    if (limite === 0) {
        secuencia = [0];
    } else {
        // Bucle para generar los siguientes números.
        while (true) {
            // Calcula el siguiente valor sumando los dos últimos.
            let siguienteValor = secuencia[secuencia.length - 1] + secuencia[secuencia.length - 2];
            
            // Si el siguiente valor supera el límite, se detiene.
            if (siguienteValor > limite) {
                break;
            }
            
            // Añade el nuevo valor a la secuencia.
            secuencia.push(siguienteValor);
        }
    }
    
    // ... Lógica para mostrar resultados ...
}

Finalmente, clasifica el resultado en categorías como "Bajo peso", "Peso normal" o "Sobrepeso".

function ejecutarIMC() {

Un palíndromo es una palabra o frase que se lee igual de izquierda a derecha que de derecha a izquierda. Para verificarlo, el programa primero "limpia" tu texto: lo convierte a minúsculas y elimina todos los espacios y signos de puntuación. Luego, crea una segunda versión de este texto limpio, pero invertida. Si el texto limpio y el texto invertido son idénticos, la frase es un palíndromo.
